# Quotation System

## Project Overview
This project is a **Quotation System** with **semi E-commerce functionality**.  
Users can browse products, select quantities, calculate prices, and generate quotations.
This project was developed as part of an **Internship Program** 

## Features
- Product management
- Select products and quantities to create quotations
- Automatic price calculation (subtotal, discount, VAT)
- Generate file to pdf

## Tech Stack
### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js
- RESTful API

### Database
- MySQL 

### DevOps / Tools
- Dockerfile
- Docker Compose
- nginx
